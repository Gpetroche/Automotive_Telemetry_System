En esta carpeta se encuentran cuatro subcarpetas con el software de la version 1.0V desarrollado para el TFG "Desarrollo de un sistema de telemetria para automoción":
-ECU_Receiver_V1.00: Contiene los códigos fuente de la unidad de control receptora. El archivo "ECU_Receiver_V1.00.ino" y el archivo "ECU_Receiver_main_V1.00.h" conforman el módulo de recepción de datos para el sistema de telemetría, siendo fundamentales para la comunicación entre los nodos de comunicación.
-ECU_Sender_V1.00: Contiene los códigos fuente de la unidad de control receptora. El archivo "ECU_Sender_V1.00.ino" y el archivo "ECU_Sender_main_V1.00.h" conforman el módulo de recepción de datos para el sistema de telemetría, siendo fundamentales para la comunicación entre los nodos de comunicación.
-GUI_MAIN_Python: Contiene los códigos fuente de la interfaz gráfica de usuario para interpretar y mostrar los datos en tiempo real. Por falta de tiempo, esta GUI no está acabada. Queda pendiente la parte de graficar los datos recibidos.
-GUI_Submodulos_en_desarrollo_PYTHON: Contiene los códigos fuente de funciones o utilidades en desarrollo pendiente de implementar en el GUI_MAIN_Python.

Versión de Python 3.11.0.0 64-bit
